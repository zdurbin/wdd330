Rewire your brain:

  • Design first for mobile instead of desktop
  • Things do not scale down very well.
  • It is easier to scale things up from small to big rather than big to small
  • Number of mobile devices are 3 to 5 times the number of desktop/laptop machines
  • The simple guideline is whatever you are doing, do mobile first.
  • Responsive does not mean mobile first
  • Better experience across all devices
  • Addresses constraints of mobile/bandwidth
  • Future friendly
  • Desktop computer sales are declining since 2011
  • Start small and scale up
  • Concentrate on content and functionality

    How to Improve UX by Understanding Users:

  • Be there, Simple and easy to use UI systems improve users experience
  • Be useful, Ensure you are giving the user what they came for.
  • Be quick. Simplify navigation to get the user to where they want to be
  • Ensujre ease of use for touch screen usability in both single and double handed holding.
  • Ensure the site is light in weight so it loads quickly
  • reduce image useage wherever possible
  • Use svgs or webfonts for images to improve loading speeds
  • Minify CSS and JavaScript
  • Gzip files to eliminate unnecessary data.

    Better Mobile UX with Strategic Menu Design:

  • ensure that navagation is easy to reach for all users. Pay attention to mobile users reach areas.
  • Placement always matters when you think about UX designs

    Designing Intuitive and Mobile Friendly Forms

  • Reduce the cogitive load wherever possible.
  • Try using float labels for field info
  • Only gather information that is absolutely needed.
  • Break large forms into smaller sections and add a progress bar.
  • Enable autofill on forms if possible.
  • Show the users password rather then auto hide it but ensure that the user can hide it if they so choose too.
  • Take into conisderation what information is being asked for and provide the appropriate keyboard.
  • consider unique ways to allow the user to input information. Especially with things like Credit card info
  • Use links for phone numbers for the company phone number to ensure the ease of use for the user.

    Designing a Better Mobile Advertising UX

  • Understand how ads are displayed on mobile devices
  • If ads are going to be displayed on mobile devices keep them small and simple.
  • Find creative ways to display ads such as using windows and background ads to make the ad a part of the content without distracting from the content.
  • Do not use popup or popover ads ever!
  • Give users what they came fore not a page full of ads.
  • Make ads unobtrusive. Content first ads always last.
  • Place yourself in the users shoes and thig about the ad placement. Would you be okay with this if you were the viewer?
